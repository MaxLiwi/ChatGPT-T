# ChatGPT Tauri
ChatGPT 桌面客户端，基于 [chatgpt-web](https://github.com/Chanzhaoyu/chatgpt-web)。

__该客户端仅支持openapi方式使用__

## 预览

![cover2](docs/s2.png)

## 介绍

### 开发

```bash
pnpm i

# 开发
pnpm tauri dev

# 打包
pnpm tauri build
```
