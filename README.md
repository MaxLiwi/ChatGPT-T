# ChatGPT Tauri
ChatGPT 桌面客户端，基于 [chatgpt-web](https://github.com/Chanzhaoyu/chatgpt-web)。

__该客户端仅支持 open-api 的方式使用__

## 介绍

### 开发

```bash
pnpm i

# 开发
pnpm tauri dev

# 打包
pnpm tauri build
```
